# Vivid Style Guide

Lively and engaging. Adds sensory details, strong verbs, vivid imagery.

## Characteristics

- **Language**: Concrete, sensory, specific
- **Verbs**: Strong, active, precise
- **Imagery**: Visual, tactile, evocative
- **Rhythm**: Dynamic, varied pacing

## When to Use

- Storytelling content
- Product descriptions
- Content that feels dry or boring
- When you want to hook readers

## Polish Approach

### Do
- Replace abstract concepts with concrete examples
- Use strong, specific verbs
- Add sensory details where appropriate
- Create visual imagery
- Vary sentence rhythm for impact

### Don't
- Overwrite or become purple prose
- Add fluff for decoration
- Lose clarity in pursuit of vividness
- Force metaphors where they don't fit

## Example Transformations

### Abstract to Concrete
**Before**: "The system performed poorly under load."

**After**: "Under heavy load, the system buckled—response times ballooned from 50ms to 8 seconds, and error rates spiked to 23%."

### Weak to Strong Verbs
**Before**: "The company made improvements to the product."

**After**: "The company overhauled the product, reimagining every feature from the ground up."

### Add Sensory Detail
**Before**: "Debugging this issue was difficult."

**After**: "Debugging this issue felt like hunting a ghost—traces appeared in logs only to vanish on closer inspection, leaving nothing but dead ends and frustration."

### Create Visual Imagery
**Before**: "Data flows through the pipeline."

**After**: "Data streams through the pipeline like water through a series of locks—transformed at each stage, filtered, enriched, and finally released as a polished result."

## Strong Verbs List

| Weak | Strong |
|------|--------|
| make | craft, forge, build, create |
| get | obtain, secure, capture, grasp |
| show | reveal, unveil, display, demonstrate |
| use | leverage, harness, wield, deploy |
| fix | repair, mend, resolve, rectify |
| start | launch, initiate, kick off, spark |
| end | conclude, terminate, culminate, close |
| improve | enhance, elevate, refine, optimize |
| change | transform, revolutionize, overhaul, shift |
| move | propel, drive, surge, flow |

## Prompt Additions

When using vivid style, add to polish prompt:

```
Style: Vivid - Lively and engaging
- Replace abstract concepts with concrete examples
- Use strong, specific verbs (avoid weak verbs + adverbs)
- Add sensory details where appropriate
- Create visual imagery readers can picture
- Vary sentence rhythm for dynamic pacing
- Show, don't tell
- Balance vividness with clarity
```
